# ユーザーフローチェック観点

画面遷移・ユーザーフローをUX観点でチェックする際の確認ポイント。

## 優先チェック項目

### 基本フロー
- [ ] ハッピーパス（正常系）が明確か
- [ ] エラーパス（異常系）が定義されているか
- [ ] キャンセル・戻るパスが存在するか

### 重要操作
- [ ] 削除・送信などの重要操作に確認ステップがあるか
- [ ] 不可逆な操作が明示されているか

### 完了・フィードバック
- [ ] 操作完了後のフィードバック画面・メッセージがあるか
- [ ] 次のアクションが明示されているか

### エラー回復
- [ ] エラー発生時にどこに戻るか定義されているか
- [ ] 入力内容が保持されるか
